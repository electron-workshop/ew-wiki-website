# Electron Workshop Wiki

This is the repository for the website that deploys the Electron Workshop Wiki. It is powered with Quartz v4 and deployed on Netlify.
🔗 Quartz Documentation: https://quartz.jzhao.xyz/

## Quick Start
The wiki is held in **/quartz/content** and the home page comes from **index.md**

The title above the search bar/button comes from **quartz.config.ts**
